# chat-app
Chat app build using Nodejs, Reactjs ( with hooks ) and Socket.io

## Start Server
Inside server folder, run command
##### > `yarn install`

After that run
##### > `yarn run server`

This will start your app server

## Start Client Server
Inside client folder, run command
##### > `yarn install`

After that run
##### > `yarn start`

This will start your app client server
